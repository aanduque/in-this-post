About jQuery In This Post

This plugin aims to provide your site guests with an easy way to navigate between your post topics and to easily meseare the post contents by its titles and subtitles.

Files Included

This plugin, as the name suggests, requires jQuery to work and it's not included. The download file you get from CodeCanyon include minified and non-minified versions of both the JavaScript code (use to develop the plugin functionality) and the CSS code (use to add its style).

Installation

To install this plugin you only need to include jQuery to your page, the main plugin JS and CSS files (the minified version is better), using the following scripts. So paste the 'dist' directory in the file your downloaded in your projects directory and give it a proper name, like 'in-this-post'. Now you can load the scripts and css.

 

<script type='text/javascript' src="in-this-post/jquery.inThisPost.js"></script>
<link rel="stylesheet" href="in-this-post/jquery.inThisPost.css">

 

Now we have to initialize the plugin passing our setup, to do that use the following code, for exemple:

 

$('.post').inThisPost({
    startingLevel: 'h3',
    subItems: false
  });

 

As you can see, you need to apply the plugin to the container where our posts resides, in this case '.post'.

A full list of the supported options follows in the next section.

Full list of Options

var options = {
      
      // This option lets you add a "extra space" in the scrolling
      // Default: 50 (px)
      offset: 50,
      
      // Use this option to define the starting level of titles to get.
      // Remeber that this also sets the sublevel to get, so if you set startingLevel as h2
      // the sublevel will be set to h3.
      // Default: 'h2'
      startingLevel: 'h2',
      
      // Here you can choose if you want to display subitems also.
      // Like said in the option above, if your startingLevel was h2, the subitems loaded
      // will be h3 and so on.
      // Deafult: true (can also be false)
      subItems: true,
      
      // This option is used to define the header element so we can decide when to display
      // the summary bar
      // Default: '.post-header, .entry-header'
      pageHeader: '.post-header, .entry-header',
      
      // This option lets you change the title used on the start of the bar
      // Default: 'In this Post:'
      title: 'In this Post:',
      
      // Use this options to select the comment block, if you want that to be displayed
      // You can also set to false, if you want to hide the comments section
      // Default: '#comments' (can also be set to false)
      comments: '#comments',
      
      // If you select the comments in the option above, you can also set the label
      // Default: 'Comments'
      commentsLabel: 'Comments',
      
      // This options let you select where you want the bar to appear
      // can be set to 'top' or 'bottom'
      // Default: 'top'
      position: 'top',
      
};
